The goal of this dashboard is to help users:

ğŸ“ˆ Key Visuals & Insights

1. CTR % & Category Performance

Displays the overall CTR %.

Highlights categories contributing to the highest performance.


2. Impressions by Month (Treemap)

Shows impression distribution across months.

Useful for identifying seasonal trends (e.g., high impressions in July, September).


3. Total Impressions Over Time

Bar charts showing impressions by month and year.

Helps track long-term growth or decline.


4. Device Analysis

Displays the total number of devices (e.g., 1500).

Shows their contribution to overall impression data.


5. Geographic Analysis

Average latitude and longitude for targeted regions.

Useful for understanding regional focus.


6. Interactive Filters

Year slider

Month checkboxes

Category filters

These tools enable dynamic exploration of the dataset.
